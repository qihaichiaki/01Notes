Linux 各种指令：

手动登录
ssh username&//ip


ls                      在当前目录显示文件名
	 -l     显示更详细的属性信息(ls是命令，-1....选项)  简写 ll
	 -a    显示所有的文件，包括隐藏文件       （Linux下是有隐藏文件/文件夹的方式，可以让文件以.开头）文件 = 内容数据 + 属性数据
	-l -a (-al)组合一起就是组合功能                       隐藏文件可以隐藏一些配置文件
	-i     查看文件的inode编号
	后跟绝对路径 表示在绝对路径下查看 相对路径即就是以当前路径为参考点去查找
	*指定文件查看

pwd                 查看当前路径                   每个目录存在两个隐藏文件 .表示当前路径(有点类似于this)    ..表示上级路径(帮助我们进行路径切换cd ..)
				/ 路径分隔符  注意，Windows是反斜杠    第一个/是根目录
cd                    进入一个路径
	cd 绝对路径/相对路径	普通用户而言：/home/XXX: 就叫做XXX这个用户的家目录，对于root用户而言就是在/root下
	cd~回到默认路径
	cd - 回到上一次所处的路径中

touch               创建一个普通文件
				stat 文件名 	ACM时间：A访问时间
mkdir               创建一个目录/文件夹
	-p 一串路径   创建一串路径（在当前目录下）
	tree 路径  将当前目录下用树状表示
		tree  需要在root权限下安装    yum install -y tree

rmdir	删除一个目录 （只能删除一个空目录）
rm  	删除一个普通文件
	-r 干掉一个有东西目录 表示递归
	-rf 强制删除，不许询问   -f强制                                          注意：Linux没有回收站，删了就是删了 rm -rf /  这个不能执行，全部删除
	* 前面加文件名 后面加后缀 表示删除特定的文件 没有加任何东西就表示在该目录下的所有了
	-i 提示删除

man	后面跟指令 询问命令的用法 即查询手册   退出按q
	云服务器下补全  yum install -y man-pages

cp	文件名    目标路径  copy 拷贝（有可能存在权限问题）
	同样如果不是拷贝普通文件 加 -rf即可 递归和强制

mv	移动普通文件  剪切到当前目录    后面跟路径
	文件名 新文件名    （跟的不是路径，那就是重命名）
	
nano 	Linux记事本 ~基本代码
		yum install -y nano  安装

提取转移均可用：
	>输出重定向   每次会覆盖							
	>>追加重定向 加入，不会覆盖，会添加
	| 管道 传送资源的 即将待处理资源给下一步指令

cat	打印文件内容
	不加文件 就读取键盘缓冲区，打印到屏幕上
	<  输入重定向     可以 cat < 文件名  和cat 文件名 一样的功能
	自然 这里也可以对文件进行操作 不存在一样的可以进行创建
	-n 显示行号
	tac 反着打印
	注意：一次性打完内容

echo	字符串 > 普通文件名     带文件名，打印到这个文件里 不加文件 打印到控制台
	作用：传输内容，文件有就直接打印，没有就创建。 本来应该显示到显示器中的内容被写入了文件中-- 

	  
more	阅读长文本 可以使用这个  -n  n可以表示对应的行号  
	注意：不能往上翻阅

less	阅读长文本 但是可以进行上下翻阅 ctrl b可往回翻阅	
	/行数   查找行数...	

head	文件名
	默认拿前10行   -n表示拿前多少行

tail	文件名
	默认拿后10行 -n表示拿后多少行

head 和 tail 可以利用管道处理文件中间部分打印

date 查看系统时间
	格式化：
	+%Y 年
	+%Y:%m月
	d日 H小时 M分钟 S秒
*
           +%s  从1970年1/1日00:00开始到现在的秒数  时间戳
	该数据单项递增，具有唯一性
 	后面加上 -d @时间戳  可以转化为正常时间

cal Linux下的日历
	-1当前月

*find	搜索路径 -name  文件名
	从搜索路径开始 以名字搜索目标文件

which	指令
	在系统路径中，查找特定指令的路径，不执行
	
alias	名字='指令'
	给指令起别名    只在本次登录有效，想长期有效需要配置脚本

grep	'文本行' 文件名	行文本过滤器
	在该文本寻找该文本行的所有行
	可以和管道和重定向组合进行批量化处理
	-n 显示在原文本的行号
	-v 去掉包含文本行的所有行
	-i 忽略大小写  默认下是对大小写敏感的

whereis  名字  在系统各个路径下找名字相关的

clear        清屏信息

xargs	指令
	将管道的输出结果，作为命令行参数传给指令
	将字符串转化成命令行参数 传给指令 作为选项使用

--打包和压缩：
	打包：将东西放在一起
	压缩：采用某种压缩算法，压缩它的存储空间
        便于传输，或者归档
	市面上各种后缀的压缩包，是非常多的解压工具，linux下也是如此
	zip/unip
	tar

zip	新文件名.zip 所压目录	打包
	只是打包了目录
	所以需要全打包就要加
	-r 即可

unzip	压缩包	解压
	-d  后面加上对应目录 会在目标路径下解压

tar	-选项czvf  新文件名 要压缩的文件  	打包压缩
	-v 显示打印压缩的过程
	-z压缩算法
	-t

	-xzvf  解压目标压缩文档
	-C 后面加上路径 在目标路径上解压和解包

bc	计算器
	quit退出

uname      -a查看Linux的体系结构和内核版本
	体系结构 硬件的x86_32...
	-r内核版本

--体系结构：英特尔 x8066   x86(x86_32) x86_64(x64)






热键：ctrl c	终止前端进程
          ctrl z	暂停
          ctrl d              快速退出
          tab		命令自动补齐
           上下翻         查看历史命令
          history

关机命令
          shutdown    	

理念：Linux下，一切皆为文件 显示器，键盘，普通文件     其read和write 方法是不一样的
	指令相当于就是一个可执行程序
	安装软件的过程其实就是将exe文件拷贝到某个路径















uname -r 查看linux内核版本信息

vmstat   报告关于内核线程、虚拟内存、磁盘、陷阱和 CPU 活动的统计信息    top cpu占用

sar 主要帮助我们掌握系统资源的使用情况，特别是内存和CPU的使用情况

stat 用于显示文件的状态信息

top 查看cpu资源使用状态

netstat 查看网络连接状态

free 查看内存资源状态

df 查看磁盘分区资源状态

grep选项中，-E选项可以用来扩展选项为正则表达式；

   $表示匹配文件末尾，字符需要在$之前表示以字符结尾  a$表示以a结尾

^表示匹配文件起始，字符需要在^之后表示以字符起始  ^a表示以a起始
