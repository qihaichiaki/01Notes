enum Sex
{
    枚举的可能取值;
}

enum Sex s = 取值;
创建变量，和结构体类似。

枚举的可能取值都是赋予了值的。
按照顺序，从上到下默认取值 0，1，2，3
如果一开始在一个值赋予，那么从这个值往下默认就是从这个基础上往下加即可。

枚举里面的取值，为枚举常量，初始化可，出了枚举就不能对其进行修改。
和#define的功能类似，但是枚举具有检查，美观，直观。
便于调试。（看到的和实际的一样的）

联合体（共用体）的关键字 union

union Un
{
    char c;
    int i;
}
union Un u;
sizeof(u)  -->大小为4个字节
联合体来说，同一时间只会用一个。
即，内存为其中最大的一个即可。
可以理解为一个人，人的身份不一样，或者一个对象的不同属性，但都是该对象的。

typedef union Un
{
    
}Un;
自定义类型。
联合体的大小也遵循对齐内存分配。至少是最大成员的大小，是要最大对齐数的整数倍，数组的对齐数就是其类型。
位顺序类似于小端 低地址在地位，高地址在高位
